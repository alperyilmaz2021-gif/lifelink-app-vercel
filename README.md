# LifeLink-App
A hospital-to-hospital medical courier web app that manages organ availability, hospital uploads, delivery requests, driver assignments, and real-time order status tracking. Designed to improve reliability, prioritization, and transparency in urgent medical transport.
# Disclaimer
LifeLink is an educational prototype developed for learning and demonstration purposes only.
It is not intended for production or commercial use.
It is not certified for clinical use and must not be used to support real-world medical decisions or handle real patient data.

# How To Run in Terminal:
pip -q install flask

unzip -o lifelink_app_demo.zip -d lifelink_app_demo

cd downloads

cd lifelink_app_demo

python3 app.py

Open the proxy button if Terminal shows it, or use https://localhost:5000/
